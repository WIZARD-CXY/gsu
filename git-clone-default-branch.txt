
当我们克隆一个仓库的时候，默认将会检出 master 分支。

如果我们想克隆的时候，默认检出其他的分支，比如 next 分支的话。我们需要修改远程裸
仓库的 HEAD 文件。

假设远程仓库的位置是 192.168.2.100:~/gsu.git，我们可以这样做：

$ ssh username@192.168.2.100 'cd gsu.git; git symbolic-ref HEAD refs/heads/next'

这样，当我们克隆仓库的时候，就会默认检出 next 分支了。

如果远程仓库托管在 github.com 的话，那么可以在管理仓库的界面中，选择要默认检出的
分支（相比之下，更加简单）。
